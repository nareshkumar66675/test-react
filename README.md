* [ReactJS] - A JavaScript library for building user interfaces


### Installation

App requires [Node.js](https://nodejs.org/) v8+ to run.
Install the dependencies and devDependencies.

```sh
$ cd test-react
$ npm install
```
### To Execute

```sh
$ npm run start
```
### Testing

```sh
$ npm run test
```
### Testing with Code Coverage

```sh
$ npm run cc
```
